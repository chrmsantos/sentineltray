# Modelos locais

## Objetivo

Use estes arquivos como referencia para criar os arquivos locais no computador do usuario.

## Como usar

1. Copie config.local.yaml para:
   %USERPROFILE%\sentineltray\config.local.yaml
2. Edite os campos comentados e reinicie o aplicativo.
3. state.json eh apenas um exemplo do formato de historico local.

## Observacoes

- Nao inclua credenciais reais nos modelos.
- Os valores sao ficticios e servem apenas como exemplo.
